# BugHunter 🎮

Un petit jeu React où vous cliquez sur des bugs avant qu'ils ne disparaissent.

## Fonctionnalités
- Timer (30s)
- Système de score
- Apparition aléatoire des bugs

## Tech
- React
- Hooks

## Règles du jeu
Cliquez sur les bugs qui apparaissent à l'écran avant qu'ils ne disparaissent pour marquer des points. Le jeu dure 30 secondes. Bonne chasse !
